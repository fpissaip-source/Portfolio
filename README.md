# Issa Portfolio

Interactive React, GSAP and Three.js portfolio with a scroll-driven 3D character scene.

## Local development

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm start
```

The visual foundation is based on Moncy Yohannan's portfolio project and is used with the creator's permission. Original source: https://github.com/MoncyDev/Portfolio-Website
